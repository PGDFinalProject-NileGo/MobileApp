# NileGo - Mobile App

Flutter-based Android app for a bicycle sharing system.

## Features
- User authentication
- Bike location map
- Bluetooth unlock/lock
- Wallet & billing
- Ride history

## Setup
1. Install Flutter SDK
2. Run `flutter pub get`
3. Configure Firebase (add google-services.json)
4. Run `flutter run`

## Tech Stack
- Flutter 3.x
- Firebase Auth
- Firestore
- Google Maps
- Flutter Blue Plus (Bluetooth)
